# わすれもの - 忘れ物防止リスト

スマホ最適化された忘れ物防止チェックリストアプリです。

## スマホで開く方法

### 方法1: ローカルサーバー経由（推奨）

1. **PCでサーバーを起動**
   ```powershell
   # このフォルダで以下のコマンドを実行
   python -m http.server 8000
   ```
   または
   ```powershell
   # Node.jsがインストールされている場合
   npx -y http-server -p 8000
   ```

2. **PCのIPアドレスを確認**
   ```powershell
   ipconfig
   ```
   「IPv4 アドレス」をメモ（例: 192.168.1.10）

3. **スマホのブラウザで開く**
   - スマホとPCを同じWi-Fiに接続
   - スマホのブラウザで以下にアクセス:
   ```
   http://[PCのIPアドレス]:8000/index.html
   例: http://192.168.1.10:8000/index.html
   ```

### 方法2: GitHub Pagesで公開

1. GitHubにリポジトリを作成
2. このフォルダの内容をプッシュ
3. Settings → Pages で公開
4. 発行されたURLにスマホからアクセス

### 方法3: ファイル転送

1. `index.html`、`style.css`、`script.js`をスマホに転送
2. スマホのファイルアプリで`index.html`を開く
3. ブラウザで表示

## 使い方

1. **項目をチェック**: リスト項目全体をタップ
2. **項目を追加**: 右下の「＋」ボタンをタップ
3. **項目を削除**: 各項目の「×」ボタンをタップ
4. **全部チェック**: 「🎉 全部チェック」ボタンをタップ
5. **リセット**: 「↺ リセット」ボタンをタップ

## 機能

- ✅ 行全体タップでチェック
- 📱 スマホ最適化UI
- 💾 ローカルストレージで自動保存
- 🎨 視覚フィードバック
- 📐 レスポンシブ対応
